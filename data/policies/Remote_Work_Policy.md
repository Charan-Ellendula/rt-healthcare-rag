# Remote Work Policy
Company: RT Healthcare
Version: 2.0

---

# 1. Purpose

Establish secure remote work standards.

---

# 2. Security Requirements

- VPN required
- MFA mandatory
- Encrypted devices only
- Screen lock enabled
- No public WiFi without VPN

---

# 3. Data Protection

Employees must ensure PHI is not exposed in remote environments.

---

# 4. Monitoring

Remote access activity may be monitored.
