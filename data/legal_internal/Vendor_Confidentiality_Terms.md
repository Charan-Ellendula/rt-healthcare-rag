# Vendor Confidentiality Terms
Company: RT Healthcare
Classification: Confidential

---

## 1. Purpose

Defines confidentiality obligations for vendors accessing RT Healthcare systems or data.

---

## 2. Confidential Information

Includes:

- PHI
- PII
- Business data
- Technical documentation
- Financial records

---

## 3. Vendor Obligations

Vendor must:

- Protect confidential information
- Use data only for contractual purposes
- Implement security controls
- Notify of breaches within 24 hours
- Allow security audits

---

## 4. Data Security Requirements

Vendor must:

- Use encryption at rest and in transit
- Implement access controls
- Maintain security monitoring
- Conduct vulnerability testing

---

## 5. Subcontractors

Vendor must:

- Ensure subcontractors comply
- Obtain written agreements
- Remain liable for subcontractor actions

---

## 6. Termination

Upon termination vendor must:

- Return or destroy all data
- Certify destruction in writing
- Cease all access immediately

---

## 7. Liability

Vendor may be held liable for:

- Data breaches
- Regulatory penalties
- Contractual damages
