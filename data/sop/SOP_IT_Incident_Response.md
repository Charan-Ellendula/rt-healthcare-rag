# SOP: IT Incident Response
Company: RT Healthcare
Owner: Security Operations
Version: 2.0

---

# 1. Purpose

To define structured response process for security incidents.

---

# 2. Incident Classification

Severity 1 – Data breach or major outage  
Severity 2 – High-impact disruption  
Severity 3 – Moderate issue  
Severity 4 – Minor issue  

---

# 3. Response Phases

1. Detection
2. Containment
3. Eradication
4. Recovery
5. Post-Incident Review

---

# 4. Reporting

- Executive notification for Severity 1
- Regulatory notification (if required)
- Root cause documentation

---

# 5. Evidence Handling

- Preserve logs
- Maintain chain of custody
- Document forensic findings
